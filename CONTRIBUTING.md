# Contributing

## Development Workflow

1. Install dependencies with `npm install`
2. Compile with `npm run compile`
3. Run tests with `npm test`
4. Start the Extension Development Host from VS Code with `F5`

## Code Rules

- Keep code comments in English
- Prefer small modules over growing `extension.ts`
- Keep explanation output short and structured
- Preserve workspace-local cache behavior

## Pull Request Checklist

- Compile succeeds
- Tests pass
- README or docs are updated when behavior changes
- New commands or settings are documented
