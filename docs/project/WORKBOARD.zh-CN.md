## 稳定约定

- 源代码注释使用英文。
- 面向用户的解释保持精简中文。
- `README.md` 和标准文档文件名默认使用英文。
- 中文镜像使用 `.zh-CN.md` 后缀。
- 远程调用前优先使用本地缓存和轻量分析。
- 导入知识和生成缓存限制在当前工作区。
- 远程预处理默认需要用户主动开启，并遵守信任、排除和大小限制。
- API Key 只保存在环境变量中，不提交到仓库。
- 新功能必须保留过期任务取消和源编辑器焦点。
- 用户可见行为变化需要测试，并同步更新中英文文档。
