# Stage 17 Summary

## Milestone

- Stage: 17
- Date: 2026-04-14
- Status: Completed

## Root Cause

- Clicking inside the explanation panel could still disrupt the active source-editor context, which made selection watching feel unstable during normal use.
- The new API-driven wordbook selector was still pruning too aggressively for common users because the practical retention ratio remained too low.
- Wordbook preprocessing still ran as one large batch, so first-use latency and jump-reading behavior were both worse than necessary.
- The explanation page had become visually noisy because the wordbook and suggested-question chips were mixed directly into the main explanation surface.

## What Was Done

- Kept a stable tracked source editor in the extension so webview focus changes no longer clear or replace the monitored code context.
- Stopped ordinary same-file explanation refreshes from canceling background wordbook preprocessing.
- Changed wordbook retention to keep all beginner candidates, about 85% for intermediate, and about 70% for expert users.
- Switched local fallback selection from hard filtering to audience-aware ranking so only the bottom slice is removed.
- Updated the remote candidate-selection prompt to target those retention ratios explicitly and avoid over-pruning.
- Split wordbook preprocessing into fast chunks of about 20 symbols, lowered remote batch budgets, and wrote partial cache updates after every chunk.
- Added recent-selection-based chunk prioritization so repeated reading in one code region pulls that region earlier in the remaining wordbook queue.
- Split the webview into a cleaner explanation page and separate wordbook page, and removed suggested-question chips from the explanation UI.

## Outcome

The extension now behaves much closer to the intended reading workflow: the panel no longer destabilizes monitoring, the wordbook keeps far more useful terms for normal users, preprocessing starts faster through chunked low-cost batches, and the UI separates explanation reading from wordbook browsing cleanly.
